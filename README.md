# JetpackSolarSystem

Solar System App:

- It is an app that serves popular news related to space and also displays a cool animation based in Jetpack Compose

Build With:

- Dagger Hilt
- Jetpack Compose
- Kotlin
- Retrofit
- MVVM Architecture
